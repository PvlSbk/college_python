Название программы: password_generator
Версия: 1.0

Описание программы:
    Программа представляет собой генератор случайных паролей с заданными параметрами с возможностью хранения полученных
    паролей в специальном файле архиве паролей формата txt.
    Порядок работы и функционал:
    После запуска программы, появляется меню, где предлагается выбрать необходимое действие:
        1. Создать новый пароль - запускается конструктор паролей
        5. Вывести архив паролей - выводятся все пароли, записанные в архивном файле паролей
        6. Очистить архив паролей - запускается очистка файла архива паролей
        7. Завершить работу программы
    После запуска конструктора паролей, появляется новое дополненное меню,
    где предлагается выбрать необходимое действие:
        1. Создать новый пароль - запускается конструктор паролей
        2. Изменить длину пароля - корректирует длину пароля
        3. Изменить сложность пароля - корректирует сложность пароля, оставляя длину прежней
        4. Создать новый пароль не изменяя длину и сложность - генерирует новый случайны пароль, не изменяя
        длину и сложность предыдущего
        5. Вывести архив паролей - выводятся все пароли, записанные в архивном файле паролей
        6. Очистить архив паролей - запускается очистка файла архива паролей
        7. Завершить работу программы

Структура программы:
    Программа состоит из 4 основных блоков main, check_array, ask_user, password_gen.
    Каждый блок, для удобства рефакторинга кода вынесен в отдельную программу
        main - в блоке осуществляется запуск программы с помощью функции run(), вывод пароля, запись нового пароля в
    архивном файле паролей
        ask_user - блок ввода информации от пользователя
        check_array - блок содержит функции для проверки массива с данными
        password_gen - блок содержит генератор случайных паролей

Техническое задание:
    Напишите программу, которая генерирует случайные пароли заданной длины и сложности.

    Пользователь должен иметь возможность выбрать:
    - типы символов, используемых в пароле (цифры, буквы верхнего и нижнего регистра, специальные символы и т.д.)

    Генератор случайных чисел в Python (модуль random) может быть полезным инструментом для создания случайных паролей.

    - Python 3.x
    - Консольный интерфейс
