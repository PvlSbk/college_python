Название программы: password_generator
Версия: 1.0

Описание программы:
    Программа представляет собой генератор случайных паролей с заданными параметрами. После создания первого пароля,
    программа предлагает выбрать дальнейшие действия, такие как:
    1. Создать пароль заново
    2. Изменить только длину пароля оставив сложность неизменной и вывести новый пароль на экран
    3. Изменить только сложность пароля оставив длину пароля неизменной и вывести новый пароль на экран
    4. Сгенерировать новый пароль не изменяя длину и сложность
    5. Завершить работу программы.

Структура программы:
    Программа состоит из 4 основных блоков main, check_array, ask_user, password_gen.
    main - в блоке осуществляется запуск программы с помощью функции run()
    ask_user - блок ввода информации от пользователя
    check_array - блок содержит функции для проверки массива с данными
    password_gen - блок содержит генератор случайных паролей

Техническое задание:
    Напишите программу, которая генерирует случайные пароли заданной длины и сложности.

    Пользователь должен иметь возможность выбрать:
    - типы символов, используемых в пароле (цифры, буквы верхнего и нижнего регистра, специальные символы и т.д.)

    Генератор случайных чисел в Python (модуль random) может быть полезным инструментом для создания случайных паролей.

    - Python 3.x
    - Консольный интерфейс
